Overview

Python 3D PIC scaffold in src/. Modules are designed for a future C++ port: explicit pushers, staggered Yee EM grids, cell-centered electrostatic grids, and prescribed external E/B fields with polar-transform geometry.

Class YeeGrid

Staggered FDTD Maxwell solver with CIC deposit/gather.

Class ElectrostaticGrid

Poisson solve, cell-centered E, periodic/anode/reflecting BCs.

Class Pushers

Boris, relativistic Boris, Vay, Higuera–Cary velocity updates.

Class ElectricFields

Analytical and file-backed prescribed E(r, t) with polarization.

Typical Driver Loop

One PIC timestep: deposit → solve fields → gather → push → move → boundaries.

for step in range(nsteps): t = step * dt grid.zero_rho() # or zero_currents for EM for p in particles: x, y, z = p.get_position() grid.deposit_rho_cic(x, y, z, p.get_charge()) grid.solve_fields() # ES: solve_fields() # EM: step_fields(dt) for p in particles: x, y, z = p.get_position() E = grid.gather_e_cic(x, y, z) # + efield.at(pos, t) if external B = grid.gather_b_cic(...) # EM only; + bfield.at(...) v = Pushers.vay(p.get_velocity(), E, B, p.get_charge(), p.get_mass(), dt) p.set_velocity(v) p.move(dt) grid.reflect_particle(p) # if reflecting BC

Particle

src/Particle.py

Macro-particle with position, velocity, charge, mass, and species kind. Velocity updates are handled externally by Pushers.

Particle(kind, x, y, z, vx, vy, vz, q, m)

Creates a particle. pos and vel are NumPy 3-vectors stored internally.

move(dt: float) -> None

Explicit position update: pos += vel * dt.

get_charge()set_charge(q)

get_kind()set_kind(kind)

Explicit particle velocity integrators. All pushers accept lab-frame E, B (3-vectors), charge q, mass m, and timestep dt.

Returns updated velocity 3-vector. Relativistic pushers default to c=1 (normalized units).

YeeGrid

src/YeeGrid.py

Staggered Yee grid for full EM PIC. Field components live on faces/edges; guard cells support periodic, anode (PEC), or reflecting (PMC-style) boundaries.

YeeGrid(nx, ny, nz, dx=1, dy=1, dz=1, ng=1, boundary="periodic", c=1.0)

Allocates staggered Ex, Ey, Ez, Bx, By, Bz, Jx, Jy, Jz, rho with guard layers.

boundaryBehavior
periodicWrap fields in guard cells
anodePEC walls: tangential E = 0, normal B = 0
reflectingPMC-style: normal E = 0, tangential B = 0; particle reflection
  • step_fields(dt) — one FDTD half-step/leapfrog cycle
  • update_b(dt) / update_e(dt) — individual Maxwell updates
  • apply_boundaries() — enforce wall/periodic guards
  • zero_fields() / zero_currents()
  • cfl_dt_limit(safety=0.99) — suggested timestep from grid spacing
  • deposit_rho_cic(x, y, z, q)
  • deposit_j_cic(x, y, z, vx, vy, vz, q)
  • gather_e_cic(x, y, z) -> ndarray
  • gather_b_cic(x, y, z) -> ndarray
  • position_in_domain(pos) — wrap or clamp per BC
  • reflect_particle(particle) — specular reflection at walls

ElectrostaticGrid

src/ElectrostaticGrid.py

Cell-centered electrostatic grid: deposit ρ, solve Poisson for φ, compute E. Supports FFT (periodic) or sparse direct solve (anode/reflecting).

ElectrostaticGrid(nx, ny, nz, dx=1, dy=1, dz=1, ng=1, boundary="periodic", eps0=1, anode_potential=0)

Fields: rho, phi, Ex, Ey, Ez on cell-centered mesh with guards.

  • solve_poisson() — φ from ρ
  • compute_e_field() — E from −∇φ
  • solve_fields() — both steps
  • zero_rho() / zero_fields()
  • deposit_rho_cic(x, y, z, q)
  • gather_e_cic(x, y, z) -> ndarray
  • gather_fields(x, y, z) -> (Ex, Ey, Ez)
  • reflect_particle(particle)

ElectricFields

src/ElectricFields.py

Prescribed spacetime-dependent E(r, t). Wave sources are authored in a fixed local frame (+z propagation). Spatial rotation into the PIC grid uses ElectricFields.transform only — no time-dependent frame rotation. Shares wave geometry helpers with MagneticFields via field_frame.

MethodDescription

uniform(amplitude)Constant E vector
sinusoidal_linear(E0, omega, k_magnitude=..., wavelength=..., psi=0)Linear pol, sine carrier, local frame
sinusoidal_elliptical(E0, omega, psi=0, delta=0, ...)Elliptical pol; delta=0 → linear
plane_wave_local(E0, omega, ...)Cosine carrier, local frame
plane_wave_from_direction(...)Returns PolarTransformedField
plane_wave_incident(theta, phi, ...)Spherical incidence angles
gaussian_pulse_local(E0, omega, center=..., width=..., psi=0, delta=0)Gaussian-enveloped cosine pulse, local frame
gaussian_pulse_from_direction(...)Returns PolarTransformedField
gaussian_pulse_incident(theta, phi, ...)Oblique Gaussian pulse via spherical angles
gaussian_pulse(...)Legacy lab-frame Gaussian pulse
linear_ramp(ramp_rate, axis="z")Time-ramped offset
from_csv(path) / from_hdf5(path, group)File-backed fields
transform(source, frame)Static polar transform wrapper

Polarization notes

  • delta = 0 — linear polarization (default)
  • delta = π/2, psi = π/4 — circular polarization

Legacy lab-frame APIs

sinusoidal(amplitude, ...), plane_wave(...), gaussian_pulse(amplitude, wavevector, ...) — fixed lab-frame amplitude vectors and envelope geometry.

at(pos, t=0.0) -> ndarray
on_grid(x, y, z, t=0.0) -> (Ex, Ey, Ez)

Combine sources with + (returns ElectricFieldsSum). Works with PolarTransformedField as well.

WaveFrame & Polar Transform

src/field_frame.py

Static local-to-lab geometry shared by electric and magnetic prescribed fields. The frame does not rotate in time; temporal behavior is in wave phase and elliptical parameters.

resolve_k_magnitude(wavevector=, k_magnitude=, wavelength=)
local_wavevector(k_magnitude) -> [0, 0, k]
evaluate_polarized_wave_local(phi, amplitude, polarization_kind, psi, delta, waveform)
normalize_envelope_width(width) -> (3,)
gaussian_envelope_local(r_local, center, width) -> float
evaluate_gaussian_pulse_local(r, t, amplitude, omega, wavevector, center, width, ...)

Used by both ElectricFields and MagneticFields for local-frame sinusoidal/elliptical waves and Gaussian pulses. The Gaussian envelope is axis-aligned in local coordinates; PolarTransformedField rotates both the envelope sampling point and the field vector into the lab frame.

WaveFrame.from_basis(k_direction, polarization=None, origin=None)
WaveFrame.from_spherical(theta, phi, pol_angle=0, origin=None)
WaveFrame.identity(origin=None)

Properties: e1, e2, k_hat (lab-frame basis vectors).

  • position_to_local(r_lab) / position_to_lab(r_local)
  • vector_to_lab(v_local) / vector_to_local(v_lab)
  • wavevector_lab(k_magnitude)
PolarTransformedField(source, frame) # alias: TransformedField

Evaluates source at local coordinates and rotates the vector into the lab frame (works for E or B):

r_local = Rᵀ (r_lab − origin)
F_lab   = R F_local(r_local, t)

Attributes: .source, .frame, .at(pos, t).

Superposition via + dispatches to ElectricFieldsSum or MagneticFieldsSum based on the operand type.

Example: Electrostatic Particle Push

Uniform background E plus self-consistent grid field on an electrostatic grid.

import sys
sys.path.insert(0, "src")

import numpy as np
from Particle import Particle
from ElectrostaticGrid import ElectrostaticGrid
from ElectricFields import ElectricFields
from Pushers import Pushers

grid = ElectrostaticGrid(32, 32, 32, dx=0.1, boundary="periodic")
efield = ElectricFields.uniform([0.0, 0.0, 1e5])

p = Particle(kind="electron", x=1.6, y=1.6, z=1.6,
               vx=0.0, vy=0.0, vz=1e5, q=-1.602e-19, m=9.109e-31)

dt = 1e-12
for step in range(100):
    t = step * dt
    grid.zero_rho()
    pos = p.get_position()
    grid.deposit_rho_cic(*pos, p.get_charge())
    grid.solve_fields()

    E = grid.gather_e_cic(*pos) + efield.at(pos, t)
    v = Pushers.boris(p.get_velocity(), E, np.zeros(3), p.get_charge(), p.get_mass(), dt)
    p.set_velocity(v)
    p.move(dt)

Example: Electromagnetic Push

Yee grid with an oblique guide field via polar transform.

import numpy as np
from YeeGrid import YeeGrid
from MagneticFields import MagneticFields
from field_frame import WaveFrame
from Pushers import Pushers

grid = YeeGrid(64, 64, 64, dx=1e-3, boundary="periodic")

local_B = MagneticFields.uniform([0.0, 0.0, 0.5])
frame = WaveFrame.from_spherical(theta=np.deg2rad(20), phi=0.0)
bfield = MagneticFields.transform(local_B, frame)

dt = grid.cfl_dt_limit()
# ... deposit, step_fields(dt), then per particle:
E = grid.gather_e_cic(x, y, z)
B = grid.gather_b_cic(x, y, z) + bfield.at(p.get_position(), t)
v = Pushers.boris_relativistic(p.get_velocity(), E, B, p.get_charge(), p.get_mass(), dt)

Example: Oblique Laser via Polar Transform

Author a wave in local coordinates, rotate into the PIC cube with a static frame.

import numpy as np
from ElectricFields import ElectricFields
from field_frame import WaveFrame

k0 = 2 * np.pi / 800e-9
omega = 2 * np.pi * 3e14

local = ElectricFields.sinusoidal_elliptical(
    E0=1e10,
    omega=omega,
    k_magnitude=k0,
    psi=np.pi / 4,
    delta=np.pi / 2,   # circular polarization
)

frame = WaveFrame.from_spherical(
    theta=np.deg2rad(30),   # incidence polar angle from +z
    phi=np.deg2rad(45),     # azimuth in xy-plane
)

laser = ElectricFields.transform(local, frame)

pos = np.array([1e-6, 2e-6, 0.0])
E_ext = laser.at(pos, t=1e-15)

Example: Oblique Gaussian Laser Pulse

Author a finite Gaussian pulse in local coordinates (envelope along local axes, propagation along local +z), then rotate into the PIC cube with a static frame.

import numpy as np
from ElectricFields import ElectricFields
from field_frame import WaveFrame

k0 = 2 * np.pi / 800e-9
omega = 2 * np.pi * 3e14

local = ElectricFields.gaussian_pulse_local(
    E0=1e10,
    omega=omega,
    k_magnitude=k0,
    center=(0.0, 0.0, 0.0),
    width=(5e-6, 5e-6, 10e-6),   # w0, w0, z_R in local frame
    psi=0.0,
)

frame = WaveFrame.from_spherical(
    theta=np.deg2rad(30),
    phi=np.deg2rad(45),
)

pulse = ElectricFields.transform(local, frame)
# or: pulse = ElectricFields.gaussian_pulse_incident(
#     E0=1e10, omega=omega, theta=np.deg2rad(30), phi=np.deg2rad(45),
#     k_magnitude=k0, width=(5e-6, 5e-6, 10e-6),
# )

pos = np.array([1e-6, 2e-6, 0.0])
E_ext = pulse.at(pos, t=1e-15)

Example: Oblique Mirror / Solenoid B

Mirror-field profile authored along local +z, rotated into the lab frame.

import numpy as np
from MagneticFields import MagneticFields
from field_frame import WaveFrame

local = MagneticFields.mirror_local(B0=0.5, scale_length=2.0)
frame = WaveFrame.from_spherical(theta=np.deg2rad(15), phi=np.deg2rad(30))
bfield = MagneticFields.transform(local, frame)

pos = np.array([0.1, 0.0, 0.2])
B_ext = bfield.at(pos, t=0.0)

Example: Paired Oblique E + B Wave

For vacuum EM waves, B is related to E by B = (k̂ × E) / ω (SI units). Build E from the transform API, then set B amplitude and pol angle accordingly.

import numpy as np
from ElectricFields import ElectricFields
from MagneticFields import MagneticFields
from field_frame import WaveFrame

omega = 2 * np.pi * 3e14
k0 = omega / 3e8
E0 = 1e8
B0 = E0 / 3e8   # |B| = |E| / c for plane wave in vacuum

frame = WaveFrame.from_spherical(theta=np.deg2rad(25), phi=np.deg2rad(10))

efield = ElectricFields.transform(
    ElectricFields.sinusoidal_linear(E0=E0, omega=omega, k_magnitude=k0),
    frame,
)
bfield = MagneticFields.transform(
    MagneticFields.sinusoidal_linear(B0=B0, omega=omega, k_magnitude=k0, psi=np.pi / 2),
    frame,
)

pos = np.zeros(3)
t = 0.0
E_ext = efield.at(pos, t)
B_ext = bfield.at(pos, t)

Example: Field Superposition

Combine bias, file-backed, and transformed sources with +.

from ElectricFields import ElectricFields
from field_frame import WaveFrame

bias = ElectricFields.uniform([0.0, 0.0, 1e4])
file_field = ElectricFields.from_csv("data/electric_uniform_z.csv")

local_pulse = ElectricFields.sinusoidal_linear(
    E0=5e9, omega=2e15, k_magnitude=1e7,
)
frame = WaveFrame.from_spherical(theta=0.2, phi=0.0)
pulse = ElectricFields.transform(local_pulse, frame)

efield = bias + file_field + pulse
E_total = efield.at(particle.get_position(), t=step * dt)
